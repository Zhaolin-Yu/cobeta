# Workspace shape: creation

Creative production with explicit checkpoints (drafting, review, polish).

## When to propose

- The work produces a single deliverable (paper, post, deck, video)
- Iteration matters; checkpoints between drafts are valuable
- Reference material informs the work but isn't the focus

## Suggested stages

| id | name | purpose |
|---|---|---|
| 01-frame | frame | Define audience, message, success |
| 02-draft | draft | First pass; quantity over polish |
| 03-revise | revise | Self-critique + targeted rewrites |
| 04-finalize | finalize | Camera-ready, packaged, shipped |

## Tags

Often `writing`, `paper`, `talk`, etc.
